# CONTRIBUTING

```
Bienvenue sur le projet
```

Ce projet recherche des contributeurs pour tous types de tâches : améliorations, correction de bugs, nouvelles fonctionnalités.

## Comment contribuer ?


Pour commencer, nous recommendons de consulter la liste des [issues]( https://github.com/dev1consultingit/projet/issues)   

- Votre première contribution pourrait par exemple être de créer une issue de type "1ere contribution" en ajoutant dans la liste des auteurs du fichie readme.md ( https://github.com/dev1consultingit/projet#readme ) votre nom d’utilisateur github.  
- Un contributeur « fork » le projet source puis soumet ses contributions via des pull requests.  
- Les pulls requests sont ensuite revues et traitées. L'issue associée est mise à jour en conséquence.
